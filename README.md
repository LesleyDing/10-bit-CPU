# 10-bit-CPU
### 10-bit CPU

1. Open the .circ file with logisim circuit simulator
2. Right click the RAM to load test file
3. Operation support:
  1. AND
  2. OR
  3. NOT
  4. ADD
  5. MOV
  6. SLL
  7. SRL
  8. SUB
  9. ADDI
  10. SUBI
  11. MOVI
  12. BNEZ
  13. NOP
  14. HALT
